sCanadian

Developed sCanadian, a web application using QuaggaJS to scan product barcodes and identify Canadian-made items. It uses Ollama to provide AI-powered recommendations which suggests Canadian alternatives for non-local products.

View Dorahacks View Presentation View Github

Summary

The design prompt was to solve a Canadian problem. We immediately thought of the current political situation with tarriffs incoming from the USA. We thought of building a tool that Canadians could easily use while shopping to ensure that they purchase Canadian products only. From then, sCanadian was born.

Key Features:

  • Barcode scanning: using the device's camera
  • Product identification: determines whether the food product is Canadian or not
  • Canadian alternatives using AI: provides 3 Canadian alternatives if a product is not Canadian
